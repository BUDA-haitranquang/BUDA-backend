JDK 1.8.0 is required for the execution of this program 
For Windows User:
- JDK 1.8.0 is packaged with the executable files.
- After installing proper JDK, "Final.exe" will runnable.
For Linux User:
- Install JDK 1.8 by command:
sudo apt-get install openjdk-8-jdk
- After installing JDK, fire up your most trusted sh, bash and run:
$> java -jar Final.jar
